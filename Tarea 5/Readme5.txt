1. Asegúrate de tener PHP instalado en tu computadora.


2. Abre una terminal o consola y navega hasta la carpeta del proyecto. Por ejemplo:

cd "C:\Users\emill\OneDrive\Desktop\Tarea 5"


3. Ejecuta el servidor embebido de PHP con este comando:

php -S localhost:2424

4. Abre tu navegador web y escribe la siguiente dirección:

http://localhost:2424

5. Ya podrás ver la página principal del portal y acceder a las distintas secciones desde el menú de navegación.
